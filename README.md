# OneTimePassword

Short description and motivation.


## Installation
Add this line to your application's Gemfile:

```ruby
gem "one_time_password"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install one_time_password
```
